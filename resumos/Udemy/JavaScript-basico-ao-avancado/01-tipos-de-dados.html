Tipos de Dados

O que é Tipos de Dados?

Essencialmente, o JavaScript é uma linguagem de tipagem dinâmica, o que significa que não é necessário declarar o tipo de uma variável quando ela é criada. No entanto, entender os diferentes tipos de dados é crucial para programar eficazmente e evitar erros comuns.

Tipos Primitivos

  1. Number: Representa tanto inteiros quanto números de ponto flutuante. 
    let numero = 123;  //inteiro
let pi = 3.14159;  //ponto flutuante
  2. Strings: Usado para representar dados textuais. Pode ser criada usando aspas simples, duplas ou crases. 
    let saudacao = "Olá, mundo!";
let resposta = 'Sim';
let texto = `O valor é ${numero}`;
  3. Boolean: True e False. O tipo booleano só tem dois valores possíveis (verdadeiro e falso). É frequentemente usado em testes lógicos. 
    let maiorIdade = true;
let menorIdade = false;
  4. Undefined: Uma variável que não teve um valor atribuído a ela é do tipo undefined. 
    let naoDefinido;
console.log(naoDefinido);  //undefined
  5. BigInt: é usado para representar números inteiros muito grandes que ultrapassam o limite seguro para o tipo Number. São escritos com um "n" no final para diferenciá-los do Number. 
    let grandeNumero = 123456789123456789123456723456789n;

Importante: Compreender esses tipos é fundamental para o controle eficaz do comportamento do programa e para evitar erros comuns de tipo, especialmente em uma linguagem de tipagem dinâmica como JavaScript.
