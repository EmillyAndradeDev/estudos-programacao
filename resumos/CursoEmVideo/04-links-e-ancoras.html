Links Externos

O que é um link externo? É um link que direciona para outro site, um site que não é meu.

Tag básica para links externos

Link que abre nova aba

→ Parâmetro target="_blank"

  • target="_blank" fica dentro da tag <a href=""></a>
  • Ou seja, a tag para links externos deve ser: <a href="" target="_blank"></a>

→ Parâmetro target="_self"

  • Tag que abre o site na própria aba em que estamos usando.

→ Parâmetro rel=""

  • Para links externos, usamos também o parâmetro rel="external", além do parâmetro target="_blank"
  • Ou seja, a tag para links externos deve ser: <a href="seuLinkAqui" target="_blank" rel="external">Texto que aparece no lugar do link para o usuário</a>

Links internos

Para usar links internos é necessário ter mais páginas no nosso projeto.

O parâmetro target blank é opcional nesse caso, porque o usuário não vai para outro site, ele continua no seu e pode retornar via link para a página inicial.

→ Você pode usar apenas a tag padrão de link: <a href="seuLinkAqui">Texto que aparece no lugar do link para o usuário</a>

→ Caso queira usar algum parâmetro target nesse caso, use:

  • target="_self"
  • Esse parâmetro indica para o navegador que a página deve ser aberta na própria página que está sendo navegada no momento. Mas não é obrigatório usar.

Usando links internos de outra pasta

Nesse caso é necessário informar o caminho.

→ Exemplo: <a href="nome-da-pasta/nomeDoArquivo.html">Texto que aparece no lugar do link para o usuário</a>

Links para download

Nesse caso temos dentro da tag <a href=""></a> o parâmetro download type="" que varia para cada tipo de arquivo.

O valor de type="" não tem nada a ver com o nome do arquivo. Ele deve ser sempre o MIME type oficial do formato, que segue o padrão categoria/subtipo.

→ o atributo type é opcional. Ele só serve como dica do tipo de conteúdo que o navegador vai lidar no download, mas normalmente o servidor já informa isso pelo cabeçalho HTTP.

→ Pode usar apenas a palavra download, sem colocar o atributo type="", nesse caso, a tag ficaria assim <a href="linkParaODownload.formatoDoArquivo" download>Baixar arquivo</a>

→ Ou seja, Se sua intenção é download obrigatório, use sempre download. O type="" é opcional, só complementa informando o tipo MIME.
