Código-fonte / Pré-formatação

A tag <code></code> é ótima para criar uma fonte monoespaçada(com o mesmo espaçamento em todas as letras). Normalmente usamos ela para demonstrar o código em uma página web, fica mais legível. Como no exemplo abaixo:

O comando a seguir é escrito em linguagem JavaScript:

document.getElementById('teste')

Vejamos outro exemplo de uso da tag <code></code> no código abaixo na linguagem Python:

num = int(input('Digite um número')) if num % 2 == 0; print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do Programa')

Mas a tag <code></code> não organiza o código com indentação.

Para que seja considerada a indentação, usamos também a tag <pre></pre> ao redor da tag <code></code>. Como no exemplo abaixo:

        
            num = int(input('Digite um número'))
            if num % 2 == 0;
                print(f'O número {num} é PAR')
            else:
                print(f'O número {num} é ÍMPAR')
            print('Fim do Programa')

Seleciona as linhas de código que quer mexer, no teclado aperta: TAB e as linhas selecionadas vão para a direita (mais distante da margem).

Tag <pre></pre>

Tudo o que estiver dentro da tag <pre></pre> é considerado, inclusive os espaços e quebras de linha.

É uma tag de formatação de texto.
