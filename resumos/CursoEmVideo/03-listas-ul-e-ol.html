Trabalhando com Listas no HTML

Listas Ordenadas

Listas ordenadas de forma padrão (numérica).

São listas onde a ordem faz diferença.

Usamos a tag <ol></ol> e dentro dessa tag, cada item deve vir dentro de uma tag <li></li>

A partir da versão 5 do html a tag </li> tornou-se opcional. Não é obrigatório fechar a tag, mas visualmente no código fica mais organizado.

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Trocar de roupa

Listas ordenadas personalizadas

A formatação padrão da tag <li></li> dentro da tag <ol></ol>, é numeração, mas é possível personalizar e alterar.

Para alterar o formato dos itens da lista basta acrescentar type="" dentro da tag <ol>. Como nos exemplos abaixo:

→ Exemplo de formatação usando type="A"

  1. Acordar
  2. Ligar para bisa
  3. Tomar café
  4. Escovar os dentes

→ Exemplo de formatação usando type="a"

  1. Acordar
  2. Ligar para bisa
  3. Tomar café
  4. Escovar os dentes

→ Exemplo de formatação usando type="I" números romanos maiúsculo:

  1. Acordar
  2. Ligar para bisa
  3. Tomar café
  4. Escovar os dentes

→ Exemplo de formatação usando type="i" números romanos minúsculo:

  1. Acordar
  2. Ligar para bisa
  3. Tomar café
  4. Escovar os dentes

Listas ordenadas personalizadas alterando o inicio da contagem da lista com start=""

Para que a lista comece a partir do número 4 ao invés do 1, por exemplo, usamos start="4" dentro da tag <ol></ol> após o type="1".

→ Exemplo de formatação usando type="1" e start="4"

  1. Acordar
  2. Ligar para bisa
  3. Tomar café
  4. Escovar os dentes
  5. Trocar de roupa

Listas não ordenadas

Listas onde a sequência não precisa estar marcada, enumerada.

Usamos a tag <ul></ul> e dentro dessa tag, cada item deve vir dentro de uma tag <li></li>

A partir da versão 5 do html a tag </li> tornou-se opcional neste caso também. Não é obrigatório fechar a tag, mas visualmente no código fica mais organizado.

  • Pão
  • Leite
  • Arroz
  • Feijão
  • Iogurte

listas não ordenadas personalizadas alterando o type dentro da tag <ul></ul>

Para alterar o formato dos itens da lista, usamos o type="" dentro da tag <ul>.

→ As variações de type="" podem ser:

  • type="disc"
  • type="circle"
  • type="square"

Lista de Definições:

Usamos a tag <dl></dl>(definition list) para criar a lista.

Dentro da tag <dl></dl>, usamos a tag <dt></dt> (o termo que será definido).

Depois, ainda dentro da tag <dl></dl>, usamos a tag <dd></dd> (a definição do termo).

→ As tags da lista de definição ficam assim:


                <dl>
                    <dt></dt>
                    <dd></dd>
                </dl>

Exemplo Lista de Definição:

HTML:
Linguagem de marcação para a criação do conteúdo de um site.
CSS:
Linguagem de marcação para a criação do design de um site.
JavaScript:
Linguagem de programação para a criação de interatividade de um site.

→ Usar <dt></dt> e <dd></dd> nos ajuda muito com os mecanismos de busca. O google, por exemplo, quando alguém perquisa "qual é a definição de HTML?" ele valoriza os sites que têm listas de <dl></dl>, com <dt></dt> e <dd></dd>.

Usar dt e dd nos ajuda muito com os mecanismos de busca. O google, por exemplo, quando alguém perquisa "qual é a definição de HTML?" ele valoriza os sites que têm listas de dl, com dt e dd.

As listas de definição são menos usadas que as ol e ul, mas são bem úteis
