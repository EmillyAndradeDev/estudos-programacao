CSS Básico

No CSS usamos /* */ para escrever comentários no código. Assim como usamos <!-- --> em HTML.

Sintaxe do CSS

  • As propriedades do CSS devem estar envolvidas por um par de chaves { }
  • Para trabalhar com o CSS é necessário informar o seletor (a tag que está sendo estilizada).
  • Propriedades são definidas por: um nome, dois pontos e um valor.

Exemplo:

body {
    color: black;
    background-color: white;
}

Adicionando o CSS ao HTML

  • Criar uma pasta CSS no computador (Disco Local C). Com o nome CSS.
  • Dentro da pasta, pelo VSCode, criar um arquivo NomeDoArquivo.css
  • No arquivo index.html colocar as tags padrão HTML:5 e dentro da tag adiciona o link, e o VSCode vai sugerir link:css
  • Depois disso, é só mudar o link no href="" dentro da e direcionar para a minha pasta de CSS

Tipos de Seletores de CSS

  • Seletor do tipo id: identifico dentro do arquivo CSS usando cerquilha #
  • : Seletor do tipo class: identifico dentro do arquivo CSS usando um ponto .

JavaScript Básico

No JavaScrip usamos // para escrever comentários no código. Assim como usamos <!-- --> em HTML.

Adicionando JavaScript no HTML

  • Criar pasta no computador, disco local C, como já vimos em HTML e CSS.
  • Dentro do VSC: criar um arquivo index.html e um arquivo javascript.js
  • No VSC, no arquivo index, dentro da tag <body> </body> adiciona a tag <script src="NomeDoArquivo.js> </script>

Variáveis

  1. Abre no VSCode o arquivo com .js
  2. Escreva a palavra let essa palavra vai ajudar o JavaScript a identificar que o que vem depois dela é uma variável.
let nomeDaVariavel = 5;
  • O que vem depois do sinal de = é o valor da variável.
  • Um ponto e vírgula ; no final quer dizer que esse trecho de código finalizou (igual ao CSS)
let fruta = "uva";
  • Usamos aspas "" para o JavaScript entender que é um texto (uma string), da mesma forma que fazemos em Python.
