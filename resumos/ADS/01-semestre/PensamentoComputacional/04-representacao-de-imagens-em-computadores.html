  • Aplicar algoritmos simples de manipulação de imagens em Python.

Resumo

  1. Fundamentos da imagem digital
    • Uma imagem digital é composta por uma grade de pixels. Cada pixel armazena informações sobre a cor e a intensidade da luz. A combinação de todos os pixels forma a imagem completa.
  2. RGB (Red, Green, Blue)
    • Cada pixel tem três valores: R, G e B, variando de 0 a 255.
    • A mistura dessas cores gera todas as cores visíveis na tela.
    • Exemplo: (255, 255, 0) → amarelo, (0, 0, 255) → azul.
  3. Grayscale (escala de cinza)
    • Cada pixel armazena apenas a intensidade luminosa, sem cores.
    • Escala varia de 0 (preto) a 255 (branco).
    • É usado em processamento de imagens, OCR e análises médicas.
  4. Imagens Bitmap x Vetoriais
    • Bitmap (raster): composta por pixels. Boa para fotos, mas perde qualidade ao redimensionar.
    • Vetorial: baseada em fórmulas matemáticas (linhas, curvas, formas). Boa para logos e desenhos que precisam de escalabilidade.
  5. Profundidade de cor e resolução
    • Profundidade de cor: quantidade de bits usada por pixel. Ex.: 8 bits → 256 cores, 24 bits → 16 milhões de cores.
    • Resolução: quantidade de pixels em largura x altura. Ex.: 1920x1080 pixels.
  6. Aplicações práticas
    • Processamento de imagens (filtros, ajustes de brilho e contraste).
    • Reconhecimento de padrões e visão computacional.
    • Desenvolvimento de jogos e design gráfico.

Exemplo Prático

→ Python: converter uma imagem RGB para grayscale usando PIL

from PIL import Image

# Abrir a imagem
imagem = Image.open("minha_imagem.jpg")

# Converter para grayscale
imagem_cinza = imagem.convert("L")
imagem_cinza.show()

→ Resultado: A imagem será exibida em tons de cinza, mantendo sua forma e detalhes.

Exercícios Para Fixar o Conteúdo

  1. Exercício: Um pixel tem RGB (50, 100, 150). Qual seria o valor aproximado em grayscale usando a média dos canais?
      Resposta:
    • Média = (50 + 100 + 150)/3 = 100 → tom de cinza médio
  2. Exercício:Explique quando é melhor usar imagens vetoriais e quando usar bitmap.
      Resposta:
    • Vetorial: logos, ilustrações que precisam ser redimensionadas sem perder qualidade.
    • Bitmap: fotos, imagens detalhadas com variações de cor complexas.
  3. Exercício: Qual a diferença entre profundidade de cor de 8 bits e 24 bits?
      Resposta:
    • 8 bits → 256 cores possíveis.
    • 24 bits → aproximadamente 16 milhões de cores, permitindo imagens mais realistas.
  4. Exercício: Uma imagem tem resolução de 800x600 pixels. Quantos pixels no total?
      Resposta:
    • Total = 800 x 600 = 480.000 pixels
