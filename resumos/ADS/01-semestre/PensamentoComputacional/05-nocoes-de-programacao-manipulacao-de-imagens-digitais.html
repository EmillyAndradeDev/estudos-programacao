Noções de Programação: Exemplos com Manipulação de Imagens Digitais

Objetivos

  • Definir instruções para manipulação simples de dados.
  • Distinguir a estrutura de repetição for.
  • Reconhecer expressões usadas em códigos de computador.
  • Distinguir a estrutura condicional if.
  • Entender conceitos de programação dinâmica em páginas HTML utilizando PHP.

Resumo

A programação é a forma de ensinar o computador a realizar tarefas específicas. No contexto da manipulação de imagens digitais, isso envolve instruções que alteram pixels individualmente ou conjuntos de pixels, permitindo criar efeitos, ajustar cores ou modificar a dimensão de uma imagem.

  1. Manipulação singular de dados

    Cada pixel de uma imagem possui informações de cor e intensidade (por exemplo, RGB). Funções como setRed(), setGreen() e setBlue() permitem ajustar os valores de cada canal de cor de forma individual.

    → Exemplo em pseudocódigo:

    Para cada pixel na imagem:
    aumentar vermelho em 50
    reduzir azul em 20
  2. Estruturas de repetição: for

    Estruturas de repetição permitem executar a mesma instrução várias vezes, essencial ao manipular imagens, pois é preciso percorrer cada pixel.

    → Exemplo em Python:

    for x in range(largura):
    for y in range(altura):
        r, g, b = imagem.getpixel((x, y))
        imagem.putpixel((x, y), (r+50, g, b))  # aumenta vermelho
  3. Expressões em código de computador

    Expressões combinam valores, operadores e funções que produzem um resultado. Na manipulação de imagens, permitem ajustar cores, calcular médias ou converter imagens para tons de cinza.

    → Exemplo em Python:

    gray = int((r + g + b) / 3)
imagem.putpixel((x, y), (gray, gray, gray))
  4. Estrutura condicional: if

    As estruturas condicionais permitem tomar decisões no código. Podemos alterar apenas os pixels que atendam a uma condição, como os que têm vermelho acima de 100.

    → Exemplo em Python:

    if r > 100:
    imagem.putpixel((x, y), (255, g, b))
  5. Programação dinâmica em páginas HTML com PHP

    A programação dinâmica permite criar páginas web que mudam seu conteúdo com base em dados ou interações do usuário. Por exemplo, podemos usar PHP para gerar conteúdo HTML que exibe imagens manipuladas ou resultados de cálculos automaticamente.

    → Exemplo simples:

    <?php
$cor = "vermelho";
echo "<h1>A cor escolhida é $cor</h1>";
?>

    Esse código gera dinamicamente um cabeçalho HTML exibindo a cor escolhida, mostrando como PHP pode interagir com dados e modificar a página sem intervenção manual.

Aplicações práticas

  • Ajuste de cores de forma seletiva (RGB)
  • Conversão de imagens para grayscale
  • Criação de filtros simples
  • Redimensionamento e manipulação de imagens
  • Uso de loops e condições para manipular pixels de forma sistemática
  • Desenvolvimento de páginas HTML dinâmicas com PHP

Exercícios Para Fixar o Conteúdo

  1. Exercício: Aumente o valor do canal azul de todos os pixels de uma imagem em 30.

    Resposta:

    Python

    for x in range(largura):
    for y in range(altura):
        r, g, b = imagem.getpixel((x, y))
        imagem.putpixel((x, y), (r, g, b+30))
  2. Exercício: Reduza o verde em metade apenas para pixels com verde maior que 120.

    Resposta:

    Python

    for x in range(largura):
    for y in range(altura):
        r, g, b = imagem.getpixel((x, y))
        if g > 120:
            imagem.putpixel((x, y), (r, g//2, b))
  3. Exercício: Converta uma imagem colorida para grayscale usando a média dos canais RGB.

    Resposta:

    Python

    for x in range(largura):
    for y in range(altura):
        r, g, b = imagem.getpixel((x, y))
        gray = int((r + g + b) / 3)
        imagem.putpixel((x, y), (gray, gray, gray))
  4. Exercício: Crie uma página HTML dinâmica usando PHP que mostre o nome de uma imagem escolhida pelo usuário.

    Resposta:

    PHP

    <?php
$imagem = "circulo.bmp";
echo "<h1>Imagem selecionada: $imagem</h1>";
?>
